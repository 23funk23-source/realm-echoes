# Realm of Echoes — роадмап

## v0.1 (MVP) — текущая версия

- [x] 3 класса (Воин, Лучник, Маг) со своими статами и умениями
- [x] 3 босса с уникальными паттернами и фазами
- [x] 3 волны миньонов (4 вида врагов)
- [x] Прокачка за забег: опыт, уровни, рост статов
- [x] Зелья HP, перманентная смерть, экраны смерти/победы со статистикой
- [x] Генеративная музыка (мирная/боссовая) и SFX
- [x] Документация (GDD, архитектура, роадмап)
- [x] Пиксель-арт спрайты классов, автоатака
- [x] Огненный данж с порталом, уникальными мобами и боссом Повелителем Пламени
- [x] Зелья ловкости с боссов (+15% скорости атаки)
