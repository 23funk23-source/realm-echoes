# Realm of Echoes — игровой дизайн (GDD)

## 1. Концепция

Однопользовательский браузерный экшен «в духе Realm of the Mad God»:

- вид сверху, свободное движение в 8 направлениях, высокая скорость персонажей;
- **автоатака** по ближайшему врагу в радиусе (зажатая ЛКМ переключает на ручной прицел мышью) — игрок фокусируется на уклонении от плотных паттернов пуль (bullet hell);
- персонажи — пиксель-арт спрайты 16×16 (рыцарь с крылатым шлемом — Лучник, фиолетовый архимаг — Маг, багровый воитель — Воин);
- **перманентная смерть** — один забег от первого уровня до финального босса;
- быстрая прокачка внутри забега: опыт → уровни → рост статов;
- длительность полного забега: **7–12 минут**.

Целевая эмоция: «ещё один заход» — короткие забеги, честная смерть (всегда видно, от чего погиб), заметный рост силы.

## 2. Мир: движение от периферии к центру

Мир — открытая карта **12800×12800** из **концентрических колец** в масштабе realm'а RotMG: чем ближе к центру, тем опаснее. Игрок появляется в лесу у южного края и пробивается внутрь; путь от кромки до арены — несколько минут ходьбы через все кольца. Население: ~170 лесных мобов, 60 исполинов, по 18 миньонов в каждом из 5 секторов, ~1300 декораций.

```
┌──────────────────────────── ЛЕС ЭХА ───────────────────────────┐
│   волки · грибы-спорометатели · лесные духи      (спавн ↓ юг)  │
│   ┌───────────────── ЗЕМЛИ ИСПОЛИНОВ ─────────────────┐        │
│   │        циклопы · гидры · фениксы (элита)          │        │
│   │   ┌────────── ПОЯС ВЛАДЫК (5 секторов) ───────┐   │        │
│   │   │  роща │ пустоши │ пики │ руины │ берега   │   │        │
│   │   │       ┌─── АРЕНА (запечатана) ───┐        │   │        │
│   │   │       │      БЕЗУМНЫЙ БОГ        │        │   │        │
│   │   │       └──────────────────────────┘        │   │        │
│   │   └────────────────────────────────────────────┘  │        │
│   └────────────────────────────────────────────────────┘       │
└─────────────────────────────────────────────────────────────────┘
```

- **Лес Эха** (внешнее кольцо) — стартовая зона: волки, грибы, лесные духи. Слабые, дают разогнаться до 3–5 уровня.
- **Земли Исполинов** — кольцо больших существ: циклопы (кольца пуль при ударе о землю), гидры (веера с трёх голов), фениксы (быстрые, оставляют огненный след и **возрождаются один раз**).
- **Пояс Владык** — пять элементальных секторов, в каждом свой мини-босс и свои мобы: Изумрудная роща (дубоходы), Пепельные пустоши (бесы, саламандры, магмовые големы), Грозовые пики (грозовики), Лунные руины (лунные тени), Затопленные берега (тритоны). Владыки привязаны к своим секторам (поводок), мобы — к точкам обитания (аггро-радиус + блуждание у дома).
- **Запечатанный центр → Цитадель Безумного Бога** — когда падает **последний из пяти владык**, через 4 секунды **всех переносит в Цитадель** — отдельную локацию финального боя (обсидиановый зал в пустоте, как замок Оракса в RotMG). Внешний мир остаётся позади; из Цитадели нет выхода — только победа или смерть.

Ориентирование: **миникарта** в правом верхнем углу (кольца, цветные метки живых владык, точка игрока), счётчик «Владык осталось: N/5», подпись текущей зоны при переходе границы.

Смерть в любой момент — конец забега (перманентная смерть). Полный забег: **15–25 минут**.

## 3. Классы персонажей

| Параметр | Воин | Лучник | Маг |
|---|---|---|---|
| HP | 160 | 105 | 90 |
| Скорость | 265 | 300 | 285 |
| Урон за выстрел | высокий | средний | низкий |
| Темп огня | низкий | средний | высокий (×2 снаряда) |
| Дальность | **короткая** | **длинная** | средняя |
| Пробивание | нет | 1 цель | нет |
| Умение (Space/ПКМ) | **Рывок-удар** — рывок к прицелу + урон по площади | **Веер стрел** — 7 пробивающих стрел конусом | **Нова** — кольцо из 22 снарядов вокруг себя |
| Кулдаун умения | 5 с | 6 с | 7 с |

Философия: Воин — риск/награда вблизи (рывок и как атака, и как уклонение); Лучник — безопасная дистанция, но мало HP; Маг — плотный DPS на средней дистанции и лучшая зачистка толпы.

## 4. Владыки стихий (мини-боссы) и Безумный Бог

Пять владык — по одному в каждом секторе пояса. Убивать можно в любом порядке; каждый дропает зелье ловкости. Все привязаны к своим секторам поводком.

| Владыка | Сектор | HP | Паттерн |
|---|---|---|---|
| **Громовой Владыка** | Грозовые пики | 1250 | короткие телепорты-блинки, быстрые тройные залпы, кольца молний |
| **Морской Царь** | Затопленные берега | 1450 | двойная спираль + «волна прилива» (широкое медленное кольцо) |
| **Страж Рощи** | Изумрудная роща | 1550 | вращающиеся спирали, веера, призыв дубоходов |
| **Лунный Архонт** | Лунные руины | 1350 | телепорты к игроку, веера, призыв лунных теней, кольцо в ярости |
| **Повелитель Пламени** | Пепельные пустоши | 1450 | огненная спираль, «волны жара», призыв бесов |

### Безумный Бог (HP 3400 × множитель мира) — финал в Цитадели

Ждёт в отдельной локации — Цитадели. Три фазы по HP:
1. **> 66%**: одиночная спираль + прицельные веера из 5 пуль.
2. **33–66%**: вторая спираль, широкие кольца, призыв лунных теней и бесов.
3. **< 33%**: третья спираль, телепорты по залу с кольцом пуль, веера из 7, залпы чаще.

С него падает **сумка с панцирем Безумного Бога** (тир = мир+1, с бонусом HP) и открывается **портал в следующий мир**.

## 4.1 Следующие миры (NG+)

Забег бесконечен: за порталом — **точно такая же карта, но всё население сильнее** (+45% HP и +25% урона за каждый мир, накопительно). Уровень, банки и экипировка игрока сохраняются. Рекорды меряются тем, **до какого мира дошёл**. Смерть — единственный конец забега.

## 4.2 Предметы и характеристики

**Характеристики игрока**: урон, броня (плоское снижение всего входящего урона, минимум 3 проходит всегда), скорость бега, скорость атаки, макс. HP. На них влияют уровень, банки и надетые предметы — всё пересчитывается автоматически.

**Экипировка** — три слота + рюкзак на 3 места. Управление предметами:
- **клик/тап** по слоту — быстро надеть/снять;
- **перетаскивание** (мышью или пальцем) — точное перемещение между слотами с подсветкой подходящих целей;
- **выбросить** — перетащить за пределы панели (предмет падает на землю рядом и его можно поднять снова) или ПКМ по слоту на десктопе.

| Слот | Предмет | Бонус (тир T = номер мира дропа) |
|---|---|---|
| Оружие | Клинок | +8% + 6%×T к урону |
| Броня | Панцирь | +2 + 2×T брони |
| Амулет | Амулет | одно из: +HP / +скорость / +скорость атаки |

Каждый владыка дропает предмет своей стихии («Клинок Грозы», «Панцирь Пламени»…); Безумный Бог — панцирь на тир выше.

**Банки** (постоянные на весь забег): **ловкость** (+15% скорости атаки) и **защита** (+2 брони) — по одной с каждого владыки (50/50), плюс зелья HP с мобов.

## 5. Прогрессия и экономика

- Опыт за убийства; порог уровня: `25 + (уровень−1) × 18`.
- Уровень: +14 макс. HP, +7% урона, +лечение 45% от макс. HP.
- Пассивная регенерация: ~1.3 HP/с (медленная, зелья важнее).
- **Зелья HP**: шанс дропа растёт с силой моба (лес 12%, миньоны секторов 25%, исполины 55%), с владык — по 2 шт.; лечат 30 HP.
- **Зелья ловкости** (аналог dex-потов RotMG): по 1 с каждого владыки; каждое даёт **+15% к скорости атаки** до конца забега (максимум: +75%).
- **Уникальный вид мобов**: каждый из 13 видов мобов, 5 владык и Безумный Бог — отдельный пиксель-спрайт 16×16 (js/sprites.js).
- Короткая неуязвимость после получения урона: 0.6 с (защита от «слипшихся» пуль).

## 6. Музыка и звук

Полностью генеративная (Web Audio API):

- **Мирная тема**: тёплый пад (аккорды Am–F–C–G), медленное арпеджио треугольной волной, лёгкая реверберация.
- **Тема босса**: тот же гармонический каркас, но темп выше, добавляется бас-пульс — напряжение без смены стиля.
- SFX: выстрел, попадание, урон по игроку, подбор зелья, левел-ап, смерть босса.
- Клавиша M — mute. Аудио стартует по первому клику (политика браузеров).
