# Realm of Echoes

**Играть онлайн: https://23funk23-source.github.io/realm-echoes/**

Браузерный мини-рогалик в духе **Realm of the Mad God**: вид сверху, буллет-хелл, перманентная смерть, прокачка за один забег. Работает на компьютере и на телефоне (виртуальный джойстик + автоатака).

- **3 класса персонажей**: Воин, Лучник, Маг — разные статы, стиль стрельбы и активное умение.
- **3 босса** с уникальными паттернами пуль: Король Слизней, Страж Обелиска, Лич.
- **Огненный данж** (опционально): портал после второго босса, уникальные мобы, босс Повелитель Пламени и двойная награда.
- **Зелья ловкости** с боссов: +15% к скорости атаки за каждое, на весь забег.
- **Генеративная музыка** на Web Audio API — спокойный эмбиент, ускоряющийся в бою с боссом. Никаких аудиофайлов.
- Без сборки и зависимостей: чистый HTML + Canvas 2D + ванильный JS.

## Запуск

Нужен любой статический сервер (из-за политики автозапуска аудио файл лучше открывать по http):

```bash
cd realm-echoes
python3 -m http.server 8788
# открыть http://localhost:8788
```

Либо просто открыть `index.html` в браузере — тоже работает.

## Управление

| Действие | Клавиша |
|---|---|
| Движение | WASD / стрелки |
| Атака | автоматически по ближайшему врагу |
| Ручной прицел | зажать ЛКМ (перекрывает автоатаку) |
| Умение класса | Space или ПКМ |
| Музыка вкл/выкл | M |
| Пауза | P |

На тач-устройствах: левая половина экрана — виртуальный джойстик, кнопка справа внизу — умение, касание правой части экрана — ручной прицел.

## Рекорды

Топ-5 лучших забегов хранится локально в браузере (localStorage) и показывается в главном меню.

## Деплой

Игра — чистая статика, хостится на GitHub Pages из ветки `main` (корень репозитория). Любой пуш в `main` автоматически обновляет прод через минуту-две. Форкнуть и поднять свою копию: включите Pages в настройках репозитория (Branch: `main`, папка `/`).
