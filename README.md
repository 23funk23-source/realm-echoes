# Realm of Echoes

**Играть онлайн: https://23funk23-source.github.io/realm-echoes/**

Браузерный мини-рогалик в духе **Realm of the Mad God**: вид сверху, буллет-хелл, перманентная смерть, прокачка за один забег. Работает на компьютере и на телефоне (виртуальный джойстик + автоатака).

- **Мирный город Зион** — хаб, где стартуют все забеги: лечащий фонтан, доска славы и портал в игровой мир.
- **Очки славы и рейтинг**: за каждого босса начисляется слава (⭐), таблица лучших с именами — в меню и на доске славы в Зионе (онлайн-бэкенд подключается одной константой `LB_URL`).
- **Открытый мир** из концентрических колец: старт из Зиона в лес на окраине → земли исполинов (циклопы, гидры, возрождающиеся фениксы) → пять элементальных секторов с владыками → запечатанная арена в центре.
- **3 класса персонажей**: Воин, Лучник, Маг — разные статы, стиль стрельбы и активное умение.
- **5 владык стихий** (мини-боссы) + финальный **Безумный Бог в отдельной Цитадели**: когда падает последний владыка, всех переносит туда (как призыв к Ораксу в RotMG). У каждого босса — телеграфированная спец-атака (красные зоны и лучи, стены пуль с брешью), из которой нужно выбегать и кайтить; вода замедляет, огонь поджигает.
- **Редкость лута**: обычный/редкий/эпический/легендарный — цветные сумки как в RotMG, пиксельные иконки предметов; чем сложнее босс и дальше мир, тем ценнее дроп.
- **Бесконечные миры (NG+)**: с бога падает сумка с бронёй и открывается портал в следующий мир — та же карта, но мобы сильнее (+45% HP, +25% урона за мир). Рекорд — до какого мира дошёл.
- **Предметы и характеристики**: клинки/панцири/амулеты с владык, банки ловкости и защиты; урон, броня, скорость и скорость атаки пересчитываются от экипировки. Предметы можно перетаскивать между слотами (мышью и пальцем) и выбрасывать на землю — перетащив за панель или ПКМ по слоту.
- **13 видов мобов с уникальными пиксель-спрайтами**, привязанных к своим зонам (аггро-радиус, блуждание у дома).
- **Миникарта** с кольцами мира и метками живых владык.
- **Зелья ловкости** с владык: +15% к скорости атаки за каждое, на весь забег.
- **Генеративная музыка** на Web Audio API — спокойный эмбиент, ускоряющийся в бою с боссом. Никаких аудиофайлов.
- Без сборки и зависимостей: чистый HTML + Canvas 2D + ванильный JS.

## Запуск

Нужен любой статический сервер (из-за политики автозапуска аудио файл лучше открывать по http):

```bash
cd realm-echoes
python3 -m http.server 8788
# открыть http://localhost:8788
```

Либо просто открыть `index.html` в браузере — тоже работает.

## Управление

| Действие | Клавиша |
|---|---|
| Движение | WASD / стрелки |
| Атака | автоматически по ближайшему врагу |
| Ручной прицел | зажать ЛКМ (перекрывает автоатаку) |
| Умение класса | Space или ПКМ |
| Музыка вкл/выкл | M |
| Пауза | P |

На тач-устройствах: левая половина экрана — виртуальный джойстик, кнопка справа внизу — умение, касание правой части экрана — ручной прицел.

## Рекорды

Топ-5 лучших забегов хранится локально в браузере (localStorage) и показывается в главном меню.

## Деплой

Игра — чистая статика, хостится на GitHub Pages из ветки `main` (корень репозитория). Любой пуш в `main` автоматически обновляет прод через минуту-две. Форкнуть и поднять свою копию: включите Pages в настройках репозитория (Branch: `main`, папка `/`).
